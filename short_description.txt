    Для выполнения домашнего задания были созданы отдельные классы.
    
    Задание 1.
    В классе Chronometer определеы публичные методы printEverySecond(),
printMessageEveryFiveSeconds(). Методы содержат блоки синхронизации, позволяющие потокам,
которые выполняют данный код, работать синхронно и взаимодействовать между собой. В классе
определена константа START_TIME, хранящая значение момента времени старта программы. 
В процессе выполнения программы ее значение используется для расчета времени, прошедшем от 
начала сессии. 
    Класс ChronometerTest содержит main-метод, в котором создается два потока выполнения:
один поток каждую секунду отображает на экране данные о времени, прошедшем от начала запуска
программы, второй поток выводит каждые 5 секунд сообщение "Прошло 5 секунд".


    Задание 2.
    В классе FizzBuzzPrinter определены синхронизированные методы fizz(), buzz(), fizzbuzz(),
number(), которые выполняют указанные в задании проверки и выводят в консоль соответтственно 
fizz, buzz, fizzbuzz или число через запятую. Конструктор класса принимает в качестве параметра
целочисленное n - верхняя граница чисел от 0 до n, которые должны быть выведены в строку в 
результате выполнения программы.
    Программа многопоточная. В mein-методе класса Application создаются четыре потока, каждый
из которых вызывает соответствующий метод.
